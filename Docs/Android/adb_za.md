# ---------------------- #
#      安装 adb
# ---------------------- #

-1-
    到 https://adbshell.com/downloads
    下载 "adb kits" 包

-2-
    解压缩后, 里面有三个文件, (不含 readme文件 )
    将三个文件分别复制进:
    C:\Windows\System32
    C:\Windows\SysWOW64
    这两个目录中,
    安装完毕

-3- 
    到 cmd 中输入 adb, 可看见安装成功了;
