# ================= #
# 单个节点的 功能
# -1- bypass (左上角)
        这个节点变成空节点, 自己不发挥作用, 但是能起到 连线的作用
        等于 unity 中的: disable
# -2- Lock (左下)
        锁定,
        将此节点及以前的所有节点的信息, 都统一存储当当前节点中来;
        此时, 如果我们把这个节点, 和前方节点断开, 这个节点也是能独立存在,
        且显示 与断开前一样的效果;
        有点类似 "打包";
        缺点就是这样做会增大 这个节点的 存储体积; 

# -3- Display/Render (右上)
        在 视图面板中, 显色这个 节点的信息;

# -4- tamplate (右下)
        视觉上就是现实这个物体的 线框,,,, 
        然后就算我们去 display 别的 节点了, 这个开启了 template 的 节点,也能显示在 显示面板中;

# -5- info 
        信息面板;

# 节点面板: 创建节点:
在右下角窗口中, 按 tab, (或者鼠标右键), 在弹出的小面板中选择 create - box;
即可创建一个 box 节点; (场景中会多出一个 box 物体) 

双击一个节点, 可以进入它的体内, 此时还能在里面创建新的节点;

点击面板左上角的 三角形, 可以回退到 上层节点


# ------------------ #
# 节点面板: 删除节点:

# ------------------ #
# 节点的 断开
-1-
    疯狂地摇动 某个节点
-2-
    用框选 选中某条线, 按 delete
-3-
    按住 Y 键, 用鼠标划过某条线


# ------------------ #
# 节点的 排布
-1-
    全局排布: L 键, 
-2-
    局部排布: 框选中某些节点, shift + L 键;


# ------------------ #
# 节点的 对齐
按住 A 键, 鼠标左键从某个节点的 出口断点开始:
-- 向下滑, 那么后面跟随的所有节点, 都竖向排好
-- 向右滑, 那么后面跟随的所有节点, 都横向排好

注意:
    只能影响后方跟随的节点
    只能向下, 向右

# ------------------ #
# 布线 中间节点
鼠标放到一根线上, 按住 alt键, 自动多出一个 小节点; 然后可以拖动这个 小节点;
这个 小节点 的目的就是用来 规划一些 长线的排布, 以至于不那么凌乱;

这个 小节点 还可以分叉出 新得路线, 来连接 后续节点


# ------------------ #
# 多个节点 打组 box
选中多个节点, 点击窗口右上角 create network box;

删除:
    右键box, 选择删除即可
