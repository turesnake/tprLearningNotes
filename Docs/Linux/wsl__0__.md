
# 如何在 windows11 中直接访问 wsl ubuntu 目录:

\\wsl.localhost\



# ======================================= #
#    vscode 直接访问 wsl ubuntu 目录
# ======================================= #

# --
    vscode 中安装 wsl 插件

# -- 
  在 wsl ubuntu bash 中进入目标 folder,输入: `code .` 
    (如果系统没有安装 code, 会立刻当场安装)
    然后 windows11 本地的 vscode 会自动打开, 关联到 ubuntu 里的这个目录;
