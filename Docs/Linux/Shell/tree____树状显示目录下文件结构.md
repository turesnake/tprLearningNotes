使用 tree 命令以 状图 的形式 打印 当前目录下的所有文件和子目录的层次结构。
tree 命令会递归地列出目录及其内容，并以树状结构显示。

# 安装
sudo apt install tree


# tree
    打印当前目录

# tree /path/to/directory
    打印指定目录

# tree -L 2
    使用 -L 选项可以限制树的深度。例如，显示最多 2 层：

# tree -f
    使用 -f 选项可以显示完整的路径：

# tree -s
