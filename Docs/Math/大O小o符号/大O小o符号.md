以下公式的 a,b 也可替换为 u(x), v(x)
可表示一个 变量，或函数

# lim a/b = 0
    说明 a 比 b 更接近 0，
    称: a 关于 b 是 高阶无穷小量 
    或: b 关于 a 是 低阶无穷小量
    ---
    a = o(b) 

# lim a/b = +inf
    说明 b 比 a 更接近 0，
    称: a 关于 b 是 低阶无穷小量 
    或: b 关于 a 是 高阶无穷小量
    ---
    a = O(b)

# lim a/b = c != 0
    称: a,b 相互是 同阶无穷小量 

# lim a/b = 1
    称: a,b 相互是 等价无穷小量 
    (等价无穷小 是 同阶无穷小 的特殊情况)
    ---
    a ~ b
    或
    a = b + o(b)
