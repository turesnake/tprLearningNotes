# ------------------------------------ #
#    共轭
#    conjugate
# ------------------------------------ #
复数里的概念:
    z  = a + bi

那么 z 的共轭为:
    z* = a - bi

# 特性:
    z * z* = a^2 + b^2 = |z|^2 


# 用途:
    若要运算 (2+3i) / (4-5i)
    ---
    此时可用 (4-5i) 的共轭来将 分母变成 实数
