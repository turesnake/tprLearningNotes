转换矩阵 的 核心功能，是将向量（pos，dir）从一个空间，转换到另一个空间。
在已知两个空间的坐标信息，如何求解 matrix，成为了学习目标






# ----------------------------------------------#
#             对 matrix 的理解
# ----------------------------------------------#
单位矩阵：
[ 1, 0 ,0 ]
[ 0, 1, 0 ]
[ 0, 0, 1 ]
可以将其理解为，对一个三维空间 坐标系的定义。
此矩阵被拆分为三个 竖向量，分别代表这个空间坐标系的 xyz 三个单位向量。
