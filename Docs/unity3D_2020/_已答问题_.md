# --- 043 --- #
Deferred shading
# 缺点：
- 因为依赖 z-buffer, 所以不擅长表现 半透明物体
    现代硬件似乎有对策
- 不适合 multiple materials
    这会增加显存开销
- 因为将 geometric 和 lighting 分离了，基于硬件的 anti-aliasing 不再表现良好
    但是似乎存在新技术
