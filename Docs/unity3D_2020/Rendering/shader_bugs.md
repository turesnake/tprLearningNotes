# ----------------------------------------------#
#         类型写错 很难报错
#         float <--> float3
# ----------------------------------------------#
原本我们想写变量:
    float3 color;
结果写成了：
    float color;
从向量，变成了标量。
然后，我们使用简化的方法去赋值：
    color = 0.5f;

原本我们希望的行为是： 
    color = float3( 0.5f, 0.5f, 0.5f );
但由于此时 类型写错，这个赋值句被解读为 标量的赋值。并不会报错
----
最后呈现的行为是 未定意的，常见的有：彩色变成灰色
