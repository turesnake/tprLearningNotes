# Environment reflection using world-space normals

当在场景中, 使用一个 天空盒来作为 reflection source, 
此时 unity 会生成一个 "默认" 的 反射探针, 它包含 天空盒的数据.

unity_SpecCube0 就是一个 unity 自动为我们准备的 cubemap, 可在 shader 中访问.
