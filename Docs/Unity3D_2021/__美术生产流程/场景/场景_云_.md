# ================================================= #
#             体积感的 云
# ================================================= #

# -1- 体积感:
不追求云边界的 棉花丝丝感, 只追求云的 体积感
看起来这些云就是一堆 颜色正确的 巨型 有机多边形;

# -2- 运动:
除了火山口 和 爆炸产生的云, 剩下的大部分自然云, 自身形变都很缓慢, 更多是平移;
这降低了我们的制作难度
