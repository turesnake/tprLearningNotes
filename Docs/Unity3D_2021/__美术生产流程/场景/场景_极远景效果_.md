

# ========================================== #
#             长满树的 山丘
# ========================================== #
可以是海岛, 也可以是山丘远景;

类似一个山丘地形, 从地面往上拔起很多圆鼓鼓的 大致朝向上方的小圆包(树)


# -1- 地面颜色存在变化, 或 "偏黄的绿" 或 绿, 或裸露岩石泥土的颜色
    也可出现类似 白峡 的断层面

# -2- 树需要一定的体积感, 但是主要追求的是量, 一大群的那种
    每棵树自己反而可以很简单;

# -3- 树在地上最好有投影, 效果会更丰富
