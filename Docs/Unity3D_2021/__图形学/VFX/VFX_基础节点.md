# ---------------------------------------------- #
#  节点
#      Turbulance
# ---------------------------------------------- #

# Intensity:
值越大, 粒子运动速度越快, 可用一个 场来控制;
比如, 每个粒子都有当前 pos值, 用这个值计算一个 perlin 值;(也可采样)
然后以此来获得自己的 速度快慢;

# Drag:
阻力系数。更高的阻力导致对粒子速度的更强的力影响。

值越大, 感觉粒子被分得更散了;

此值为 0 时, 湍流彻底关闭;

# Frequency:
Unity 对噪声进行采样的周期。更高的频率导致更频繁的噪声变化。

这个值太大的话, 粒子会做 无规则布朗运动;

# Octaves
噪声层数。更多的八度音阶会产生更多样化的外观，但计算起来也更耗费资源。


# Roughness
Unity 用于每个八度音程的 缩放因子。 Unity 仅在 Octaves 设置为高于 1 的值时使用粗糙度。


# Lacunarity
每个连续八度音阶的频率变化率。 1 的空隙度值导致每个八度音程具有相同的频率。
