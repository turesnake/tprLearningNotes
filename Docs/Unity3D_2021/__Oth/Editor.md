# ----------------------------------------------#
#      让函数成为一个 editor 模式的按钮
# ----------------------------------------------#
public class A : MonoBehaviour{

    [MenuItem("tpr/button1")]
    static void func1(){
        ...
    }
}
    通过上述实现，可以在 unity 界面中多出一个 tpr 菜单，
    里面有一个 “button1” 按钮
    在 app 非运行模式，点击这个按钮，就能调用这个 函数
    比如用来自动化地 配置 or 生成一些东西
