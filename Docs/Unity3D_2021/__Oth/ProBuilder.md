#       新建物体 
# ----------------------------------------------#

# 方法-1-
    ProBuilder面板中, 点击 New Sharp, 
    选中几何形体, 配置好参数,
    按住 shift, 此时 scene 窗口中会出现一个 待安放的实例物体
    鼠标左键 可以放下



# ----------------------------------------------#
#       
# ----------------------------------------------#
使用 scene 中新增的 4个按钮可以编辑 已生成的 物体
选择面工具, 可以移动一个面, 

按 shift , 同时移动某个面, 会造出一个新的面
(不过此时只能沿 xyz 轴拉动)


# ----------------------------------------------#
#    按 shift 可复选: 多个点,线,面 之类的
