# --------------------------- #
#   虚拟相机 切换速度
# --------------------------- #
在场景中简单地 开启关闭某个 虚拟相机, 就能完成两个相机之间的 切换;

想要修改这个过度速度, 找到 Cinamachine Brain 组件 - "Default Blend"
    这一行最右侧, 有个 秒数 输入框, 设置小值来提高速度;

# 在 "Default Blend" 下方有个 "Custom Blends" 
    可以手动定制不同的虚拟相机的 切换模式;
    非常细腻实用;
