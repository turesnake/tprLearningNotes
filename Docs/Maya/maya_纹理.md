# ------------------------------------ #
#     如何更换 要绘制的 贴图
# ------------------------------------ #

# -1- 先打开 "Hypershade" 窗口:
    左上角选择 "渲染"
    顶部窗口:
        窗口 - 渲染编辑器 - Hypershade

# -1- 
    回去选中 模型,
    在 Hypershade 中间节点面板顶部, 点击 "为选定对象上的材质制图"

# -1- 
    节点编辑器中选择 贴图节点
    在右侧面板中更换贴图

# -1- 
    然后回到 "3D 绘制工具" 中, 就会直接在新的图上绘制了
