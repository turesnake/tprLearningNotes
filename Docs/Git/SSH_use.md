#     如何查看 win git bash 默认使用的 ssh key ?
# ----------------------------------------- #
https://stackoverflow.com/questions/18845799/show-ssh-key-file-in-git-bash

在 git bash 中输入: git-gui
然后在打开的 app 中: help - Show SSH Key

从而看到目标 ssh key
