# - Prescribe
    开(药)
    ---
    [拉丁-法-英]      "预先写下", "提前记录"

    用于描述设定规则或指示的行为。

    在医学中，prescribe 用于描述医生为患者开具药物或治疗方案的行为。这种用法强调医生在治疗之前设定的指示。

    法律和规则: Prescribe 也用于法律和规则的上下文中，表示设定某种行为或程序的规定。

    现代意义
    今天，“prescribe”不仅用于医学，还广泛应用于任何需要设定规则或指示的领域。它强调在行动之前设定明确的指导或规定的过程。
