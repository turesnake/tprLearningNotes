# ----------------------------- #
#    如何为 win11 账号设置密码
# ----------------------------- #
点击 田字格 主按钮, 点击 左下角当前账户 - "更改账户设置"
在打开的窗口中, 后撤到 "账户" 节目, 选择 登录选项, 然后就有了



# ----------------------------- #
#    如何 删除一个 win11 账户
# ----------------------------- #
点击 田字格 主按钮, 点击 左下角当前账户 - "更改账户设置"
在打开的窗口中, 后撤到 "账户" 节目,
- "其他账户"
- 找到目标账户, 删掉之

# ----------------------------- #
#    .csv 文件如何存储为 utf-8 格式
# ----------------------------- #

# excel 存储的 utf-8 编码的 .csv 文件, 会偷偷带一组 BOM 字符;
如果用 wsp 来打开这个文件, 然后再保存, 文件格式会被强制改为 ansi 格式;

# 处理方法:
再次用 vscode 打开这个文件, 点击窗口右下角的 UTF-8 按钮, 
点击 "save with encoding"
    选择
        UTF-8 with BOM

就好了
