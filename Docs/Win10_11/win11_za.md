# ----------------------------- #
#    如何查看 ip 地址
# ----------------------------- #

# -1-:
    打开 "任务管理器"
    选择第二项 "性能"
    选择 wifi
    就能看到了

# -2-:
    -- 右下角, 点击无线网络图标, 
    -- 选中当前连接的 wifi
    -- 选择 "更多 wi-fi 设置"
    -- 选择 "硬件和属性"

    就能看到了



# ----------------------------- #
#    如何为 win11 账号设置密码
# ----------------------------- #
点击 田字格 主按钮, 点击 左下角当前账户 - "更改账户设置"
在打开的窗口中, 后撤到 "账户" 节目, 选择 登录选项, 然后就有了



# ----------------------------- #
#    如何 删除一个 win11 账户
# ----------------------------- #
点击 田字格 主按钮, 点击 左下角当前账户 - "更改账户设置"
在打开的窗口中, 后撤到 "账户" 节目,
- "其他账户"
- 找到目标账户, 删掉之
