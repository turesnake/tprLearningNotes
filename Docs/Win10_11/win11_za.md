# ----------------------------- #
#    如何为 win11 账号设置密码
# ----------------------------- #
点击 田字格 主按钮, 点击 左下角当前账户 - "更改账户设置"
在打开的窗口中, 后撤到 "账户" 节目, 选择 登录选项, 然后就有了



# ----------------------------- #
#    如何 删除一个 win11 账户
# ----------------------------- #
点击 田字格 主按钮, 点击 左下角当前账户 - "更改账户设置"
在打开的窗口中, 后撤到 "账户" 节目,
- "其他账户"
- 找到目标账户, 删掉之
