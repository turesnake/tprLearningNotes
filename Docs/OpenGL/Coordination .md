坐标体系。



# ================================================ #
#           Normalized Device Coordinates ／ NDC
#                 ( 标准化设备坐标 )
# ================================================ #
xyz 三个轴 的取值范围都在 -1 ～ 1. 
坐标中心 在 世界的中心。

Vertex Shader / 顶点着色器 的 输出数据 就得遵循 NDC体系。




# ================================================ #
#              Texture Coordinate
#                  ( 纹理坐标 )
# ================================================ #
对于 2D纹理坐标：
    左下 -- (0,0)
    右上 -- (1,1)

纹理坐标的 三轴为: s,t,r
对应于     传统的: z,y,z

# ================================================ #
#          Local Space / Object Space
#         （ 局部空间坐标 ／ 物体空间坐标 ）
# ================================================ #
物体／对象 相对于 原点的 坐标。也是 物体起始坐标。



# ================================================ #
#                World Space
#              （ 世界空间坐标 ）
# ================================================ #
相对于 世界的 全局原点 会和其它物体一起，相对于世界的原点 进行摆放


# ================================================ #
#           View Space / Eye Space
#         （ 观察空间坐标 ／ 视觉空间坐标 ）
# ================================================ #
从摄像机／观察者 的角度 出发的 坐标。

#                Clip Space
