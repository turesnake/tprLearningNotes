# cd 到指定目录:
-1-
    在同一盘符中时, 直接 cd, 
    比如初始在 C 盘, 想要到 C:\AAA\BBB, 可以直接写:
        cd C:\AAA\BBB

    此时会发现, 命令行 头部发生变化, 此时可键入 "dir" 查看;

-2-
    若要跳转到另一盘符, 需要先进入目标盘:
    比如, 想要到 F:\AAA\BBB
    --
        先键入 F:
        此时会跳转到 F盘
    --
        再键入 cd F:\AAA\BBB
