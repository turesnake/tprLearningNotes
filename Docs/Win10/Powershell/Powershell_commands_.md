# 示范:
    set-location -path "D:\AAA\"

此时 ps 会进入这个目录;

# 示范:
    set-location -path "D:\AAA\" -PassThru

进入目标目录, 然后再把这个 目录path 打印出来;


# 和 win cmd 不同, 不需要事先手动切换盘符;
